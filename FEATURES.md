# 功能特性说明

## 🎯 核心功能

### 1. 微信浏览器检测 ⚠️
- **自动检测**：页面加载时自动检测是否在微信浏览器中打开
- **友好提示**：显示全屏警告遮罩，提示用户使用独立浏览器打开
- **可关闭**：用户可以点击"我知道了"按钮或点击背景区域关闭提示
- **键盘支持**：按 ESC 键也可以关闭提示

### 2. 响应式设计 📱
- **桌面端**：多列卡片网格布局（根据屏幕宽度自适应）
- **平板端**：2-3列布局
- **手机端**：单列布局，优化触摸体验
- **断点**：
  - 1200px - 大屏幕优化
  - 768px - 平板和小屏幕
  - 480px - 小屏手机

### 3. 卡片布局 🎴
- **动态网格**：使用 CSS Grid，自适应列数
- **悬停效果**：
  - 卡片上移
  - 阴影增强（橙色光晕）
  - 顶部彩色条出现
  - 图标放大旋转
  - 链接箭头右移
- **平滑过渡**：所有交互都有流畅的动画效果

### 4. 视觉设计 🎨
- **渐变背景**：深色渐变背景（紫蓝色系）
- **比特币主题色**：使用比特币标志性橙色 (#f7931a)
- **玻璃态卡片**：半透明白色卡片，现代感十足
- **动画效果**：
  - 页面加载时的渐入动画
  - 比特币图标的脉冲动画
  - 卡片的交错出现动画
  - 悬停时的流畅过渡

### 5. 预留扩展位置 ➕
- **占位卡片**：已添加 2 个占位卡片
- **样式区分**：占位卡片使用虚线边框，半透明样式
- **易于替换**：只需将占位卡片替换为实际链接即可

### 6. 性能优化 ⚡
- **DNS 预加载**：自动为外部链接预加载 DNS
- **懒加载动画**：使用 Intersection Observer 实现视口内动画
- **网络状态监测**：监听在线/离线状态
- **预留 Service Worker**：为未来离线支持做准备

### 7. 可访问性 ♿
- **键盘导航**：完整的键盘操作支持
- **语义化 HTML**：正确使用 HTML5 语义标签
- **减弱动画模式**：支持系统的减少动画设置
- **打印优化**：提供打印样式

## 🔧 自定义指南

### 修改配色
在 `styles.css` 的 `:root` 部分修改 CSS 变量：

```css
:root {
    --primary-color: #f7931a;        /* 主色调 */
    --bg-gradient-start: #0f0c29;    /* 背景起始色 */
    --bg-gradient-mid: #302b63;      /* 背景中间色 */
    --bg-gradient-end: #24243e;      /* 背景结束色 */
}
```

### 添加新卡片
在 `index.html` 的 `cards-grid` 中添加：

```html
<a href="你的链接" class="card" target="_blank" rel="noopener noreferrer">
    <div class="card-icon">🎯</div>
    <h3 class="card-title">工具名称</h3>
    <p class="card-description">工具描述</p>
    <div class="card-footer">
        <span class="card-link">访问 →</span>
    </div>
</a>
```

### 修改页面标题
在 `index.html` 中修改：
- `<title>` 标签：浏览器标签标题
- `<h1>` 标签：页面主标题
- `.subtitle`：副标题
